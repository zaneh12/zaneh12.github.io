Project Goal¶

We develop a Bayesian model to infer the latent belief distribution of traders in a prediction market using only snapshots of the standing limit order book. By modeling observed volume at each price as a Gamma-distributed function of belief mass and distance from the midprice, we recover the underlying distribution of trader beliefs without relying on order flow or trade data.

Stronger Punchline with Formal Examples¶

This model recovers latent market structure from observable book shape and provides formal, interpretable outputs that can be used to analyze market dynamics. After Bayesian inference:

  • The posterior over $ \mu_t $ yields a continuous estimate of the market-implied probability at time $ t $. Unlike midprice or VWAP, it reflects the full distribution of standing liquidity.

    Example: In a high-liquidity Polymarket contract on a U.S. election, $ \mu_t $ shifts from 0.42 to 0.51 following a key debate. This movement reflects a change in the belief-weighted depth of the market, not just a best bid/ask flinch.

  • The posterior over $ \nu_t $ allows us to distinguish between high-consensus markets (tight book around a single belief) and dispersed or adversarial markets (liquidity spread across conflicting price regions).

    Example: For an obscure geopolitical question, $ \mu_t = 0.47 $ but $ \nu_t \approx 2.5 $, indicating wide disagreement among traders despite a stable midprice. This identifies cases where the market price is not representative of strong consensus.

  • By tracking $ (\mu_t, \nu_t) $ over time, we produce belief dynamics, enabling real-time monitoring of how markets digest information.

    Example: In a sports market, $ \nu_t $ spikes during an injury announcement, indicating temporary fragmentation in trader beliefs before reconverging.

  • Across contracts, comparing $ \nu_t $ enables market classification: events with low $ \nu_t $ exhibit noisy, speculative trading; high $ \nu_t $ markets reflect expert-driven consensus.

This gives a principled, interpretable way to extract meaningful belief summaries from order book data alone, without relying on message data, market maker assumptions, or agent-based simulations.

Bayesian Inference of Belief Distributions from Limit Order Books¶

We model the standing volume $V(p)$ at each discrete price level $p$ in a binary prediction market. At each snapshot in time, we observe the set of active bid and ask orders and their associated sizes.

We assume that traders have probabilistic beliefs $b \in [0, 1]$ and place limit orders based on those beliefs. Specifically, a trader will place a bid if $b > p$ and an ask if $b < p$, with volume decaying as $p$ moves further from $b$.

We model the distribution of beliefs with a Beta CDF:

$$ B(p) = \text{CDF}_{\text{Beta}}\left(p;\ \mu \cdot \nu,\ (1 - \mu) \cdot \nu\right) $$

Here, $\mu \in (0,1)$ represents the mean belief, and $\nu > 0$ controls concentration.

We define a shape function $\psi(p)$ for the volume curve:

$$ \psi(p) = \begin{cases} (1 - B(p)) \cdot \exp\left(-\beta(p^* - p)\right) & \text{if } p < p^* \\ B(p) \cdot \exp\left(-\beta(p - p^*)\right) & \text{if } p > p^* \\ 0 & \text{otherwise} \end{cases} $$

where $p^*$ is the midprice and $\beta > 0$ controls how quickly volume decays away from $p^*$.

We assume expected volume is proportional to $\psi(p)$:

$$ \mathbb{E}[V(p)] = \alpha \cdot \psi(p) $$

To model noise and excess variation, we place a Gamma likelihood over $V(p)$:

$$ V(p) \sim \text{Gamma}\left(\kappa(p),\ \gamma\right) \quad \text{where} \quad \kappa(p) = \frac{\alpha \cdot \psi(p)}{\gamma} $$

This yields the log-likelihood:

$$ \log \mathcal{L}(\theta) = \sum_p \left[ (\kappa(p) - 1) \log V(p) - \gamma V(p) - \log \Gamma(\kappa(p)) + \kappa(p) \log \gamma \right] $$

with parameters $\theta = (\alpha, \beta, \mu, \nu, \gamma)$.

We complete the model with priors:

  • $ \alpha \sim \text{HalfNormal}(\sigma_\alpha) $
  • $ \beta \sim \text{HalfNormal}(\sigma_\beta) $
  • $ \mu \sim \text{Beta}(a, b) $
  • $ \nu \sim \text{Gamma}(s, r) $
  • $ \gamma \sim \text{Gamma}(s', r') $

Inference is performed using MCMC or variational methods. Posterior summaries of $\mu$ and $\nu$ yield interpretable estimates of the market-implied probability and the degree of belief consensus or disagreement.

# Define a grid of parameter choices: each tuple is (mu, nu, beta_param)
param_grid = [
    (0.3, 5, 5),
    (0.3, 20, 5),
    (0.5, 5, 5),
    (0.5, 20, 5),
    (0.7, 5, 5),
    (0.7, 20, 5),
    (0.5, 20, 2),
    (0.5, 20, 10),
]

# Price grid
p_grid = np.linspace(0.1, 0.9, 100)
fig, axes = plt.subplots(2, 4, figsize=(20, 10), sharey=True, facecolor='white')
axes = axes.flatten()

for i, (mu, nu, beta_param) in enumerate(param_grid):
    ax = axes[i]

    Bp = beta.cdf(p_grid, mu * nu, (1 - mu) * nu)
    midprice = 0.5

    psi_bid = np.where(
        p_grid < midprice,
        (1 - Bp) * np.exp(-beta_param * (midprice - p_grid)),
        np.nan
    )
    psi_ask = np.where(
        p_grid > midprice,
        Bp * np.exp(-beta_param * (p_grid - midprice)),
        np.nan
    )

    ax.plot(p_grid, psi_bid, color='blue', label='Bid side')
    ax.plot(p_grid, psi_ask, color='red', label='Ask side')

    ax.axvline(midprice, color='gray', linestyle=':', linewidth=1)

    # Annotate interpretation
    description = []
    if mu < 0.5:
        description.append("skewed left")
    elif mu > 0.5:
        description.append("skewed right")
    else:
        description.append("symmetric")

    if nu < 10:
        description.append("diffuse beliefs")
    else:
        description.append("tight consensus")

    if beta_param < 4:
        description.append("slow decay")
    elif beta_param > 8:
        description.append("sharp decay")

    ax.set_xlabel("Price")

from scipy.stats import beta
import numpy as np
# --- Model Parameters ---
import scipy.optimize as opt

# Define loss function: sum of squared error between model mean and actual volume
def loss_fn(params, df, midprice):
    alpha, beta_param, mu_belief, nu_belief = params

    # Prevent invalid parameter regions
    if alpha <= 0 or beta_param <= 0 or mu_belief <= 0 or mu_belief >= 1 or nu_belief <= 0:
        return np.inf

    # Compute B(p)
    df = df.copy()
    df['B(p)'] = beta.cdf(df['Price'], mu_belief * nu_belief, (1 - mu_belief) * nu_belief)

    # Compute psi(p)
    def psi(row):
        p = row['Price']
        if row['Side'] == 'bid' and p < midprice:
            return (1 - row['B(p)']) * np.exp(-beta_param * (midprice - p))
        elif row['Side'] == 'ask' and p > midprice:
            return row['B(p)'] * np.exp(-beta_param * (p - midprice))
        else:
            return 0
    df['psi(p)'] = df.apply(psi, axis=1)

    # Predicted mean volume
    df['ModelMean'] = alpha * df['psi(p)']

    # Loss = sum of squared differences
    return np.sum((df['ModelMean'] - df['Size']) ** 2)

# Initial guess for parameters: alpha, beta, mu, nu
initial_guess = [50000, 10, 0.4, 5]

# Perform optimization
opt_result = opt.minimize(
    loss_fn,
    initial_guess,
    args=(plots, midprice),
    method='L-BFGS-B',
    bounds=[(1e3, 1e7), (0.1, 50), (0.01, 0.99), (1, 50)]
)

# Extract optimal parameters
alpha_opt, beta_opt, mu_opt, nu_opt = opt_result.x

# Recalculate model with best-fit parameters
plots['B(p)'] = beta.cdf(plots['Price'], mu_opt * nu_opt, (1 - mu_opt) * nu_opt)

def compute_psi_fit(row):
    p = row['Price']
    if row['Side'] == 'bid' and p < midprice:
        return (1 - row['B(p)']) * np.exp(-beta_opt * (midprice - p))
    elif row['Side'] == 'ask' and p > midprice:
        return row['B(p)'] * np.exp(-beta_opt * (p - midprice))
    else:
        return 0

plots['psi(p)'] = plots.apply(compute_psi_fit, axis=1)
plots['ModelMean'] = alpha_opt * plots['psi(p)']
