What I Read 02-23-2023

Mutually Exciting Hawkes Process

  1. We will consider a poisson point process or counting process. We assume that we have a sequence of identically and independently distributied poisson variables over some interval. We observe some number of events. We want to talk about the behaviour of the simple point process and model it by specifying the "conditional intensity". This represents the "infinitesimal rate at which events are expected to occur around a particular time t, conditional on the prior history of the point process prior to time t."
  2. This point process is said to be "Self Exciting" if we can observe: the covariance of some point in time N(s,t) and N(t,u) for s < t < u. We then observe this "intensity" as equal to the background poisson rate, in most cases say the scoring rate of a basketball team, but this is summed over the time between the next event say, decaying over time by some kernel we specify, exponential, power law etc. Effectively, this "changes" the poisson rate in the background we specify for a small period of time. Depending on the parameter fit, we can say they score at a rate of x, but this is now x+n(t,t-n) over the interval. It is mean reverting. Basically a clustering density.
  3. Mutually Exciting Processes can be more useful when we have different types of events. Take in finance the interaction between prices of different stocks. One can assume that Coca Cola stock and Pepsi Stock are mutually exciting. The new equation is simply extended over the events from 1,..,D, for each event contributing to the intensity.
  4. I am hoping to implement this in a sports context like basketball.
